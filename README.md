# Masakin 
Smart Recipe Finder Based on Your Ingredients

Masakin adalah aplikasi pencarian resep berbasis bahan yang tersedia di dapur.  
Pengguna cukup memilih bahan yang dimiliki, dan aplikasi akan merekomendasikan resep yang relevan secara cepat dan praktis.

---

## Fitur Utama

- Pencarian resep berdasarkan bahan
- Login Google (opsional, untuk personalisasi nama & foto)
- Tampilan responsif (mobile-friendly)
- Antarmuka ringan dan cepat
- Tidak menyimpan data sensitif pengguna

---

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Firebase Authentication
- Firebase Firestore / Database

---

## Keamanan & Privasi

- Data login Google hanya digunakan untuk personalisasi tampilan.
- Tidak ada penyimpanan data sensitif di server eksternal.
- Data preferensi disimpan secara lokal di perangkat pengguna.
- Firebase rules dikonfigurasi untuk membatasi akses publik.
